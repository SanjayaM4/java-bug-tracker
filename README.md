# Bug Tracker (Java)

A simple console-based bug tracking system implemented in Java.  
Supports adding bugs, storing them in `bug.dat`, and viewing/updating bug history.

---

## You Can Run Directly Online Using GitHub Codespaces

### 1. Open Codespaces
1. Go to this repository on GitHub  
2. Click the green **"<> Code"** button  
3. Open the **Codespaces** tab  
4. Click **"Create codespace on main"**

GitHub will create a cloud-based VS Code environment with Java preinstalled.

---

### 2. Compile and Run the Program

Once the Codespace launches, run:

```bash
javac *.java
java Main
```

## To Run Locally On Your Own Machine

### 1. Ensure Java is Installed
You must have Java 8 or higher

### 2. Download the Project

#### Option A - Download ZIP
1. Click Code → Download ZIP
2. Extract the ZIP file

#### Option B - Clone Repository
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
```

### 3. Compile the Program
```bash
javac *.java
```

### 4. Run the Program
```bash
java Main
```
